# Metacommerce Junior Frontend Developer Task - Simple Notes App

Notes app wrote on pure react

## Features:

- [x] Support Markdown
- [x] Create/update/delete notes
- [x] List/gallery view
- [x] Note searching
- [x] Font style presets
- [x] Serverless
